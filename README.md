# AI Agent Platform

A modern full-stack application platform with React frontend and backend services.

## Project Structure

```
ai-agent-platform/
├── frontend/          # React + Vite + TailwindCSS frontend application
└── backend/           # Backend services (to be implemented)
```

## Frontend

The frontend is built with modern web technologies:

- **React 19**: Latest React with improved performance and features
- **Vite 7.2.2**: Lightning-fast build tool with HMR
- **TailwindCSS v4**: Utility-first CSS framework
- **ESLint**: Code quality and consistency

### Getting Started with Frontend

```bash
cd frontend
npm install
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
npm run lint     # Run ESLint
```

The development server will start at `http://localhost:5173/`
